# 🍳 ChefBot PWA - App Mobile

## Déployer sur Vercel (gratuit, 2 minutes)

### Option 1 : Via le site web (plus facile)
1. Va sur https://vercel.com et crée un compte gratuit (connecte-toi avec GitHub)
2. Crée un repo GitHub et push ce dossier dedans
3. Sur Vercel, clique "Add New Project" → importe ton repo
4. Root Directory : mets `public`
5. Clique "Deploy"
6. T'as un lien genre `chefbot-xyz.vercel.app` !

### Option 2 : Via la ligne de commande
1. Installe Vercel CLI : `npm i -g vercel`
2. Dans ce dossier, lance : `vercel`
3. Suis les instructions
4. C'est déployé !

## Installer sur ton cell

### iPhone :
1. Ouvre le lien dans Safari
2. Tape le bouton "Partager" (carré avec flèche)
3. "Sur l'écran d'accueil"
4. ChefBot est installé avec l'icône du chef !

### Android :
1. Ouvre le lien dans Chrome
2. Chrome va te proposer "Ajouter à l'écran d'accueil"
3. Accepte et c'est fait !
